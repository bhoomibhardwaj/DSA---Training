278. First Bad Version

BASIC LOGIC:

Versions are sorted like:

Good Good Good Bad Bad Bad

Use Binary Search

Take middle version

If mid is bad → search left side
If mid is good → search right side

Repeat until start == end

Return start

CODE:
public class Solution extends VersionControl {
    
    public int firstBadVersion(int n) {
        int start = 1;
        int end = n;
        
        while (start < end) {
            int mid = start + (end - start) / 2;
            
            if (isBadVersion(mid)) {
                end = mid;          // first bad is at mid or before
            } else {
                start = mid + 1;    // first bad is after mid
            }
        }
        
        return start;   // or return end (both same here)
    }
}
